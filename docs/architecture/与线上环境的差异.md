# 与线上环境的差异

> Parent: [README.md](README.md)

本地模拟器用于快速验证云脚本逻辑，不等价于真实平台环境。

## 已知差异

- 本地通过 HTTP Bridge 通信，不经过真实平台链路
- 无平台签名、鉴权和包格式校验
- Lua 脚本运行时限制与线上不完全一致（如超时阈值、内存上限）
- 网络延迟、断线重连、排队机制需要在线上额外验证
- 本地存档以 JSON 文件保存，线上平台使用服务端存储
- 道具消耗为本地模拟，不经过平台支付和库存系统
- 本地 Lua 版本为 5.3（与平台一致），但具体运行时细节可能有差异

## 建议

- 本地测试覆盖逻辑正确性和事件流
- 发布前在真实平台环境验证协议、性能、超时和存档兼容性
- 不要把本地 Bridge 配置和测试代码带入正式发布包
