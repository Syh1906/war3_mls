# MLS 云脚本本地模拟器

Rust 实现的 MLS 云脚本本地测试环境，单文件部署，内置 Web 控制面板。

用于在本地模拟平台 MLS 运行时，无需连接线上服务器即可测试云脚本逻辑。

## 技术栈

| 组件 | 说明 |
| --- | --- |
| 语言 | Rust |
| HTTP 框架 | axum + tokio |
| Lua 运行时 | mlua（Lua 5.3，静态链接） |
| WebSocket | axum 内置，原生协议 |
| Web 前端 | HTML/JS/CSS，编译时嵌入二进制 |
| 配置格式 | JSON |
