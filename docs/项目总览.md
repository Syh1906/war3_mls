## 项目定位

MLS 云脚本本地模拟器是一个 Rust 单二进制工具，用来在本地模拟平台云脚本运行环境。它支持多客户端连接、事件收发、Lua API 注入、存档读写和 Web 控制面板。

```text
云脚本作者 / 测试脚本
        |
        v
HTTP Bridge / Web 面板
        |
        v
Rust 模拟器 -> 房间线程 -> Lua VM
        |
        v
本地 JSON 存档
```

## 代码边界

| 路径 | 责任 |

## 维护原则

`origin/main` 是个人 Fork 的开发主线，可以包含自己的 docs 体系。向上游贡献时，从 `upstream/main` 新建干净分支，只挑选适合上游的代码或文档提交。
